# LS00001. Random or Not?

Given a sequence of numbers, 
return `true` if the sequence is random or `false` if it was created by a human.

## Examples

Example 1:

```plain
Input: seq = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
Output: true
```

## Constraints

- 10 <= seq <= 50
- 0 >= seq[i] <= 9
